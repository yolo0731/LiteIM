# LiteIM Qt Icon Resource Rules

Step 45 only defines the icon-resource boundary. It does not add final product icons.

Rules:

- Keep Qt client icons under this directory or a direct subdirectory.
- Use only original LiteIM assets, permissively licensed assets, or assets created specifically for this project.
- Do not use WeChat, Weixin, QQ, Telegram, Slack, Discord, or other product logos, names, icons, screenshots, or brand-identifying artwork.
- Record third-party asset license, source URL, author, and modification notes before adding the asset.
- Prefer SVG for simple UI icons and PNG only when raster output is required.
- Keep filenames descriptive, lowercase, and hyphen-separated, for example `conversation-empty.svg`.
